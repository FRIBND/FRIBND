Nuclear Physics Angular Momentum Coupling Guide

This document provides the theoretical background and practical usage guide for the angular_momentum_coupling.py utility. This tool calculates allowed final nuclear states based on quantum mechanical selection rules.

1. Theoretical Basis

The code implements the Channel Spin Coupling Scheme, which is a standard method for determining angular momentum conservation in nuclear reactions.

Notation

Symbol Meaning

πparticle\pi_{particle}πparticle​ Parity of transferred particle (Python particle input)

Notation discipline: Use lowercase sparticles_{particle}sparticle​ for the intrinsic spin of the transferred object (single nucleon, nucleon pair, or cluster). Uppercase SSS is reserved for the channel spin formed by coupling JtargetJ_{target}Jtarget​ with sparticles_{particle}sparticle​. Do not use SSS to describe the transferred object itself.

Conservation Laws

In any nuclear reaction Target+Particle→Final_StateTarget + Particle \to Final\_StateTarget+Particle→Final_State, the following quantities are conserved:

  1. Total Angular Momentum (J⃗\vec{J}J):

    J⃗final=J⃗target+s⃗particle+ℓ⃗\vec{J}_{final} = \vec{J}_{target} + \vec{s}_{particle} + \vec{\ell}Jfinal​=Jtarget​+sparticle​+ℓ

  2. Parity (π\piπ):

    πfinal=πtarget×πparticle×(−1)ℓ\pi_{final} = \pi_{target} \times \pi_{particle} \times (-1)^{\ell}πfinal​=πtarget​×πparticle​×(−1)ℓ

Coupling Scheme Used in Code

The code performs the vector addition in two steps (Channel Spin representation):

  1. Calculate Channel Spin (S⃗\vec{S}S): Couples the target spin and particle spin.

    S⃗=J⃗target+s⃗particle\vec{S} = \vec{J}_{target} + \vec{s}_{particle}S=Jtarget​+sparticle​

    Possible values: ∣Jtarget−sparticle∣≤S≤Jtarget+sparticle|J_{target} - s_{particle}| \le S \le J_{target} + s_{particle}∣Jtarget​−sparticle​∣≤S≤Jtarget​+sparticle​

  2. Calculate Final Spin (J⃗final\vec{J}_{final}Jfinal​): Couples the channel spin with the orbital angular momentum.

    J⃗final=ℓ⃗+S⃗\vec{J}_{final} = \vec{\ell} + \vec{S}Jfinal​=ℓ+S

    Possible values: ∣ℓ−S∣≤Jfinal≤ℓ+S|\ell - S| \le J_{final} \le \ell + S∣ℓ−S∣≤Jfinal​≤ℓ+S

Important: The "Particle" input to this tool represents the intrinsic properties of the transferred particle: spin sparticles_{particle}sparticle​, parity πparticle\pi_{particle}πparticle​, and isospin TparticleT_{particle}Tparticle​.

2. Application to Different Reaction Types

To use the code correctly, you must identify the correct "Particle" input based on the reaction mechanism.

A. Resonant Capture Reactions

Examples: (p,γ)(p, \gamma)(p,γ), (n,γ)(n, \gamma)(n,γ)

In these reactions, a projectile fuses with the target to form a compound nucleus (resonance).

  • Particle Input: 1/2+

B. Single-Nucleon Transfer Reactions

Examples: (d,p)(d, p)(d,p), (p,d)(p, d)(p,d), (3He,d)(^3\text{He}, d)(3He,d), (d,3He)(d, ^3\text{He})(d,3He), (n,d)(n, d)(n,d), (d,n)(d, n)(d,n), (d,t)(d, t)(d,t), (t,d)(t, d)(t,d), (α,t)(\alpha, t)(α,t), (t,α)(t, \alpha)(t,α), (3He,α)(^3\text{He}, \alpha)(3He,α), (α,3He)(\alpha, ^3\text{He})(α,3He)

In these reactions, a single nucleon is transferred between the projectile and the target. You need to input the properties of the transferred nucleon.

  • Particle Input: 1/2+

C. Two-Nucleon Transfer Reactions

In these reactions, a pair of nucleons is transferred. The total spin of this pair depends on the coupling of the two nucleons.

1. Identical Nucleon Transfer (2n or 2p)

Reactions: (p,t)(p, t)(p,t), (t,p)(t, p)(t,p) (2n transfer); (3He,n)(^3\text{He}, n)(3He,n) (2p transfer)

For the transfer of two identical nucleons (2n or 2p) in the same shell model orbit, the Pauli Exclusion Principle dictates that their total wavefunction must be antisymmetric.

  • If they are in a relative s-state (spatially symmetric, L=0L=0L=0), their spin wavefunction must be antisymmetric.
  • An antiparallel spin state for two fermions corresponds to sparticle=0s_{particle}=0sparticle​=0 (spin singlet for the transferred pair).
  • NDS Policy: For (p,t)(p, t)(p,t), (t,p)(t, p)(t,p), and (3He,n)(^3\text{He}, n)(3He,n) reactions, it is standard to assume the transferred pair is in a relative sss state (sparticle=0s_{particle}=0sparticle​=0).
  • Particle Input: 0+ for transferring sparticle=0,Tparticle=1s_{particle}=0, T_{particle}=1sparticle​=0,Tparticle​=1 particle pair

2. Neutron-Proton Transfer (1n1p)

Reactions: (α,d)(\alpha, d)(α,d), (d,α)(d, \alpha)(d,α), (3He,p)(^3\text{He}, p)(3He,p), (p,3He)(p, ^3\text{He})(p,3He)

For the transfer of a neutron-proton pair, the selection rules depend on the projectile/ejectile properties.

  • (α,d)(\alpha, d)(α,d) and (d,α)(d, \alpha)(d,α):

    • Both ddd and α\alphaα have T=0T=0T=0. Thus, only T=0T=0T=0 transfer is allowed.
    • Antisymmetry requires sparticle+Ts_{particle}+Tsparticle​+T to be odd (for L=0L=0L=0). Since T=0T=0T=0, the transferred pair must be spin triplet, i.e., sparticle=1s_{particle}=1sparticle​=1.
    • Particle Input: 1+ for transferring sparticle=1,Tparticle=0s_{particle}=1, T_{particle}=0sparticle​=1,Tparticle​=0 particle pair

  • (3He,p)(^3\text{He}, p)(3He,p) and (p,3He)(p, ^3\text{He})(p,3He):

    • Both particles have sparticle=1/2,Tparticle=1/2s_{particle}=1/2, T_{particle}=1/2sparticle​=1/2,Tparticle​=1/2.
    • Allowed transferred-pair components: (sparticle=0,Tparticle=1)(s_{particle}=0, T_{particle}=1)(sparticle​=0,Tparticle​=1) and (sparticle=1,Tparticle=0)(s_{particle}=1, T_{particle}=0)(sparticle​=1,Tparticle​=0).
    • Particle Input:
      • 0+ for transferring sparticle=0,Tparticle=1s_{particle}=0, T_{particle}=1sparticle​=0,Tparticle​=1 particle pair
      • 1+ for transferring sparticle=1,Tparticle=0s_{particle}=1, T_{particle}=0sparticle​=1,Tparticle​=0 particle pair
    • Note: Both components can contribute.

D. Cluster Transfer Reactions

Examples: (6Li,d)(^6\text{Li}, d)(6Li,d), (7Li,t)(^7\text{Li}, t)(7Li,t)

An alpha particle is transferred.

  • Particle Input: 0+

E. Inelastic Scattering

Examples: (α,α′)(\alpha, \alpha')(α,α′), (p,p′)(p, p')(p,p′)

  • (α,α′)(\alpha, \alpha')(α,α′): The alpha particle has spin 0. Since the projectile spin cannot flip, the angular momentum transfer is purely orbital (ℓ⃗\vec{\ell}ℓ). This selectively excites Natural Parity resonances (π=(−1)ℓ\pi = (-1)^\ellπ=(−1)ℓ) in the compound nucleus.

  • Particle Input: 0+

  • (p,p′)(p, p')(p,p′): The proton (s=1/2s=1/2s=1/2) can undergo spin-flip. It allows both Isoscalar (T=0T=0T=0) and Isovector (T=1T=1T=1) transitions.

    • Particle Input: 0+ for non spin-flip
    • Particle Input: 1+ for spin-flip

F. Charge Exchange Reactions

Examples: (p,n)(p, n)(p,n), (3He,t)(^3\text{He}, t)(3He,t), (6Li,6He)(^6\text{Li}, ^6\text{He})(6Li,6He)

These reactions exchange nucleon type (n ↔ p), governed by the transferred angular momentum (ΔL\Delta LΔL), spin (ΔS\Delta SΔS), and total angular momentum (ΔJ=ΔL+ΔS\Delta J = \Delta L + \Delta SΔJ=ΔL+ΔS). Parity change follows πf=πi(−1)ΔL\pi_f = \pi_i (-1)^{\Delta L}πf​=πi​(−1)ΔL.

Transition Types

  • Fermi (ΔL=0,ΔS=0,ΔJ=0,ΔT=0\Delta L = 0, \Delta S = 0, \Delta J = 0, \Delta T = 0ΔL=0,ΔS=0,ΔJ=0,ΔT=0): The emitted electron and antineutrino (or positron and neutrino) have antiparallel spins, and hence, no spin-flip, no parity change. Excites the Isobaric Analog State with Jf=JiJ_f = J_iJf​=Ji​ and πf=πi\pi_f = \pi_iπf​=πi​.

  • Gamow-Teller (ΔL=0,ΔS=1,ΔJ=1,ΔT=0,±1\Delta L = 0, \Delta S = 1, \Delta J = 1, \Delta T = 0, \pm1ΔL=0,ΔS=1,ΔJ=1,ΔT=0,±1): The emitted leptons have parallel spins, and hence, spin-flip. Selection rule: Jf=Ji,Ji±1J_f = J_i, J_i \pm 1Jf​=Ji​,Ji​±1 but except Ji=0→Jf=0J_i = 0 \to J_f = 0Ji​=0→Jf​=0. Therefore, from 0+0^+0+ targets, GT transitions selectively populate 1+1^+1+ states. Except Ti=0→Tf=0T_i = 0 \to T_f = 0Ti​=0→Tf​=0.

  • Spin-Dipole (ΔL=1,ΔS=1\Delta L = 1, \Delta S = 1ΔL=1,ΔS=1): Parity change (πf=−πi\pi_f = -\pi_iπf​=−πi​), with ΔJ=0,1,2\Delta J = 0, 1, 2ΔJ=0,1,2.

Particle Input

  • (6Li,6He)(^6\text{Li}, ^6\text{He})(6Li,6He): Pure Gamow-Teller (1+1^+1+ 6Li^6\text{Li}6Li to 0+0^+0+ 6He^6\text{He}6He projectile).

    • Particle Input: 1+

  • (p,n)(p, n)(p,n) and (3He,t)(^3\text{He}, t)(3He,t): Mixed (Fermi + Gamow-Teller).

    • Particle Input: 0+ (Fermi)
    • Particle Input: 1+ (Gamow-Teller)

4. Isospin Coupling

For transfer reactions, isospin (TTT) provides additional selection rules.

  • Single Nucleon Transfer: Transferring one nucleon (nnn or ppp) has Tparticle=1/2T_{particle} = 1/2Tparticle​=1/2.
  • Identical Nucleon Pair (2n/2p): In a relative L=0L=0L=0 state, the Pauli Principle requires sparticle=0s_{particle}=0sparticle​=0 (spin singlet for the transferred pair) and Tparticle=1T_{particle}=1Tparticle​=1 (isospin triplet).
  • Neutron-Proton Pair (np):
    • Deuteron-like (sparticle=1,Tparticle=0s_{particle}=1, T_{particle}=0sparticle​=1,Tparticle​=0)
    • Singlet (sparticle=0,Tparticle=1s_{particle}=0, T_{particle}=1sparticle​=0,Tparticle​=1)

Selection Rule: The final isospin is determined by vector addition:

T⃗final=T⃗target+T⃗particle\vec{T}_{final} = \vec{T}_{target} + \vec{T}_{particle}Tfinal​=Ttarget​+Tparticle​

Note: This tool calculates angular momentum (JπJ^\piJπ) only. Isospin selection rules must be applied separately.
